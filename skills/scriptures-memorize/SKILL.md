# Scripture Memorization

Provide tools and techniques to memorize a scripture verse.

## When to Use

Use when the user wants to:
- Memorize a specific verse
- Learn a doctrinal mastery passage
- Commit scripture to heart

## Workflow

### Step 1: Get the verse

Use `mcp__scriptures__lookup_verse` to get the exact text.

### Step 2: Break it down

Divide the verse into:
- Memorable phrases (3-5 word chunks)
- First-letter mnemonics
- Key word structure

### Step 3: Provide memorization techniques

Offer multiple approaches:
1. **Chunking**: Break into phrases
2. **First Letters**: Create acronym
3. **Visualization**: Mental images for each phrase
4. **Music**: Suggest setting to simple tune
5. **Writing**: Practice writing from memory

### Step 4: Create practice exercises

Provide:
- Fill-in-the-blank versions
- First-letter prompts
- Scrambled phrase reordering

### Step 5: Give review schedule

Suggest spaced repetition:
- Today: 5 times
- Tomorrow: 3 times
- Day 3: 2 times
- Day 7: Review
- Day 14: Review
- Day 30: Review
